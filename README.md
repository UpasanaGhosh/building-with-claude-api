# Building with Claude API 🚀

This repository documents my journey from learning the Anthropic Claude API to building **production-style AI systems**.

Instead of treating this as a tutorial dump, I approached it like an engineer:

* Every concept is implemented as a **working system**
* Projects are structured like **real-world applications**
* Emphasis is placed on **evaluation, scalability, and design patterns**

---

## 🧠 What This Repo Demonstrates

This repo goes beyond basic API usage and focuses on **building intelligent systems** using Claude:

* Prompt engineering & structured outputs
* Multi-turn conversational systems
* Retrieval-Augmented Generation (RAG) pipelines
* Tool use & function calling
* Model Context Protocol (MCP) implementations
* Agent-based architectures (planning, routing, execution)
* Evaluation frameworks for LLM outputs
* Multimodal workflows (text, images, PDFs)

---

## 📂 Key Projects

### 🗣️ Conversational AI System

A multi-turn chat system with context handling, system prompts, and response control.

### 🧪 Prompt Engineering + Evals

Experimentation with prompt strategies, including:

* Few-shot learning
* XML structuring
* Automated evaluation pipelines

### 📚 RAG Pipeline

End-to-end retrieval system:

* Document ingestion & chunking
* Hybrid search (BM25 + embeddings)
* Context-aware generation

### 🛠️ Tool-Using Agents

Extending Claude with external tools and structured workflows.

### 🔌 MCP Server & Client

Implementation of Model Context Protocol for standardized tool integration.

### 🤖 Agent Architectures

Systems that simulate reasoning workflows:

* Task routing
* Parallel execution
* Chained decision-making

### 🖼️ Multimodal Processing

Using Claude for:

* Image understanding
* PDF parsing
* Citation generation
